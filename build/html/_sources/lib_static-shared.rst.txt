=================================
Les libraries (static / shared)
=================================



STATIC 
---------------
 * Liées au moment de la compilation
 * Le code est inclus dans l'executable final.

 * Format: .lib (win) ou .a (Uniw)


SHARED
---------------

 * liée au moment de l'execution
 * Format: .dll (win)  .so (Unix)


.. note:: 
    La mise a jour de la librairie dynamique ne nécéssite pas la recompilation de l'executable
